# TreeMap

📜 Este projeto é um TreeMap interativo utilizado para visualização de dados.

## 🛠 Instalação

Para iniciar o projeto, siga os passos abaixo:

1. **Clone o projeto.**
   
2. **Execute o projeto no VSCode usando a extenção LiveServer.**


## ⭐ Tecnologias Utilizadas 

- **HTML Canvas**
- **BootStrap**
- **JavaScript**
